# Security Policy

## Reporting a vulnerability

Do not open a public GitHub issue for security-sensitive findings.

Instead, report the issue privately to:

- `security@controlplane.local`

Include:

- affected area
- reproduction steps
- impact assessment
- suggested remediation if available

## Scope

The highest-priority security surfaces are:

- authentication
- database credentials and env handling
- public customer-order views
- file uploads
- push notification flows
